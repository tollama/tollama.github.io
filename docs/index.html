Ollama for Time Series.
Run state-of-the-art foundation models locally.

A local-first forecasting daemon with a unified REST API. Pull, serve, and query Chronos, TimesFM, Moirai, Sundial, Granite TTM, and Toto through one CLI and HTTP interface. Latest updates include expanded forecast metrics (MAPE, MASE, MAE, RMSE, SMAPE), first-party LangChain tools, the tollama-forecast OpenClaw skill package, and tollama-mcp for MCP clients.

Quickstart

tollama runs a FastAPI daemon on http://127.0.0.1:11435 by default and exposes an Ollama-style API under /api. Start the daemon, pull a model, then forecast via curl or the CLI.

1) Install (from source)
python3.11 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -e ".[dev]"
2) Start the daemon
tollama serve
# check:
curl http://localhost:11435/api/version
tollama doctor
tollama info --json
3) Pull + forecast (HTTP)
# pull
curl -s http://localhost:11435/api/pull \
  -H 'content-type: application/json' \
  -d '{ "name": "chronos2" }'

# forecast (example payload)
curl -s http://localhost:11435/api/forecast \
  -H 'content-type: application/json' \
  -d @examples/chronos2_request.json

For a full run walkthrough (including per-family runtime isolation and installing all registry models), read docs/how-to-run.md.

OpenClaw users can install skills/tollama-forecast and run scripts/e2e_skills_test.sh to verify skill integration.

MCP users can install .[mcp], run tollama-mcp, and optionally register Claude Desktop via scripts/install_mcp.sh.

For evaluation workflows, /api/forecast accepts optional metrics names: mape,mase,mae,rmse,smape.

// MCP Support
Native MCP server for assistant clients.

Tollama includes tollama-mcp for Claude Code and compatible MCP clients. The stdio MCP toolchain was re-validated on February 20, 2026.

Install and run MCP
python -m pip install -e ".[mcp]"
tollama serve
tollama-mcp

Default MCP base URL is http://localhost:11435. Each tool supports optional base_url and timeout overrides.

Register in Claude Desktop
bash scripts/install_mcp.sh --base-url "http://127.0.0.1:11435"
# preview only:
bash scripts/install_mcp.sh --dry-run --base-url "http://127.0.0.1:11435"

// REST API
Forecast with
a single POST.

Send time series data with covariates and optional actuals, receive probabilistic forecasts and evaluation metrics in one response. Consistent schema across all model families.

json ¬∑ /api/forecast POST
{
  "model": "timesfm-2.5-200m",
  "horizon": 2,
  "series": [{
    "id": "store_001",
    "freq": "D",
    "target": [120, 135, 142],
    "actuals": [141, 145],
    "past_covariates": {
      "promo": [0, 1, 0]
    },
    "future_covariates": {
      "promo": [1, 0]
    }
  }],
  "parameters": {
    "covariates_mode": "best_effort",
    "metrics": {
      "names": ["mape", "mase", "mae", "rmse", "smape"],
      "mase_seasonality": 1
    }
  }
}
